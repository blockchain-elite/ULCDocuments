Let's get started

ULC Documents is a set of smart contracts in the Ethereum Blockchain that allow a moral or physical personn to virtually sign all type of documents with a blockchain level of security & reliability. This web application allow you to interact with theses smart contract to check or sign files, raw text or directly hashes. ULC Documents is decentralised, open source and can be used freely. You only pay Ethereum network costs when you want to sign something.
